INSTALL PACKAGES
- Install "Microsoft.EntityFrameworkCore" from Nuget.
- Install "Npgsql.EntityFrameworkCore.PostgreSQL" from Nuget.
- Install "Microsoft.EntityFrameworkCore.Tools" from Nuget.

MIGRATIONS
- Open Package Manager Console
- Set Default Project as your project
- Commands:
-- "add-migration"
-- "add-migration migrationname"
-- "remove-migration"
-- "remove-migration migrationname"
-- "update-database"
